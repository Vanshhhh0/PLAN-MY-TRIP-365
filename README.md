# PLAN-MY-TRIP-365
PlanMyTrip-365 is a centralized, AI-driven Tourism Guide and Travel Management Application designed to simplify, personalize, and streamline the fragmented travel planning process. Built using Java within the NetBeans IDE environment and powered by a MySQL relational database, the application connects a secure user interface to its back-end architecture via a robust JDBC controller pipeline. The platform features user authentication, custom itinerary creation, location searches for nearby infrastructure, and tailored destination package configurations (such as a multi-day Gold Package). To support seamless transaction flows and ensure data integrity, the system integrates a secure, cashless multi-channel payments gateway—supporting services like UPI, Paytm, PhonePe, and G-Pay—while utilizing automated database trigger controls to manage user profiles securely without query overhead.
